# Open EPC report

I wrote a script that extracts data from the Open EPC data and generates a pdf for it.
The report includes:
- The total number of EPCs carried out in the local authority
- A graph of the number of EPCs against property type (house, flat, massionette, Bunaglow, Park home)
- Number of records against ratings bands

I've structured the code using classes so that the code can be appended to easily.
