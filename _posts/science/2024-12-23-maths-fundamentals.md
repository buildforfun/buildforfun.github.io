# Maths fundamentals
## 1. Foundations of Mathematics

### Proof Techniques
Understanding different proof methods is essential in mathematics. The main techniques include:

- **Direct Proof**: A straightforward approach where the conclusion is established directly from the premises using logical deductions.
  
- **Proof by Contradiction**: This method assumes that the statement to be proven is false, leading to a contradiction, thus proving that the statement must be true.

- **Mathematical Induction**: Used primarily for proving statements about natural numbers. It involves two steps:
  1. **Base Case**: Verify the statement for the initial value (usually $$ n = 1 $$).
  2. **Inductive Step**: Assume the statement holds for $$ n = k $$ and prove it for $$ n = k + 1 $$.

### Set Theory
Set theory serves as a foundation for modern mathematics, particularly through Zermelo-Fraenkel set theory with the Axiom of Choice (ZFC). Key concepts include:

- **Sets**: Collections of distinct objects, considered as objects in their own right.
  
- **Functions**: A relation between sets that associates each element of a first set with exactly one element of a second set.
  - **Injections**: Functions that map distinct elements of one set to distinct elements of another.
  - **Surjections**: Functions where every element of the second set is mapped by at least one element of the first set.
  - **Bijections**: Functions that are both injections and surjections, establishing a one-to-one correspondence between sets.

- **Relations**: A subset of the Cartesian product of two sets, describing how elements from one set relate to elements in another.

## 2. Calculus

### Differential Calculus
Differential calculus focuses on the concept of the derivative, which represents the rate of change of a function. Key topics include:

- **Limits**: Fundamental to understanding derivatives, limits define the behavior of functions as they approach specific points.
  
- **Derivatives**: The derivative of a function at a point quantifies how the function's output changes as its input changes. Common rules include:
  - Product Rule
  - Quotient Rule
  - Chain Rule

- **Applications**: Derivatives are used in optimization problems, motion analysis, and modeling rates of change.

### Integral Calculus
Integral calculus deals with accumulation and areas under curves. Important concepts include:

- **Definite Integrals**: Represent the area under a curve between two points.
  
- **Indefinite Integrals**: Represent families of functions whose derivatives yield a given function.

- **Techniques of Integration**: Methods such as substitution, integration by parts, and partial fractions.

- **Applications**: Used in calculating areas, volumes, and solving differential equations.

### Real Analysis
Real analysis rigorously examines functions and sequences. Key topics include:

- **Sequences and Series**: Understanding convergence and divergence, including tests for convergence (e.g., ratio test, root test).
  
- **Continuity**: A function is continuous if small changes in input result in small changes in output.

- **Convergence**: A sequence converges if it approaches a specific limit as its index increases.

## 3. Linear Algebra

### Vector Spaces
Vector spaces are fundamental structures in linear algebra characterized by:

- **Vectors**: Elements that can be added together and multiplied by scalars.
  
- **Axioms**: Operations must satisfy properties such as closure, associativity, commutativity, identity elements, and distributive properties.

- **Bases and Dimension**: A basis is a set of vectors that spans a vector space; dimension refers to the number of vectors in a basis.

### Matrices
Matrices are rectangular arrays that represent linear transformations and systems of equations. Key concepts include:

- **Matrix Operations**: Addition, multiplication, and scalar multiplication.
  
- **Determinants**: A scalar value that provides information about a matrix's properties (e.g., invertibility).

- **Eigenvalues and Eigenvectors**: Eigenvalues are scalars associated with a linear transformation represented by a matrix; eigenvectors are non-zero vectors that change only in scale during this transformation.

## 4. Differential Equations

### Ordinary Differential Equations (ODEs)
ODEs involve functions and their derivatives. Important aspects include:

- **First-order ODEs**: Can often be solved using separation of variables or integrating factors.
  
- **Second-order ODEs**: Solutions may involve characteristic equations or particular solutions depending on initial conditions.

### Applications
Differential equations model various real-world phenomena such as population dynamics, heat transfer, and mechanical systems.

## 5. Abstract Algebra

### Groups, Rings, and Fields
Abstract algebra studies algebraic structures defined by sets equipped with operations. Key structures include:

- **Groups**: Sets equipped with a single operation satisfying closure, associativity, identity element existence, and invertibility.

- **Rings**: Structures with two operations (addition and multiplication) where addition forms an abelian group and multiplication is associative.

- **Fields**: Rings where every non-zero element has a multiplicative inverse; fields allow division (except by zero).

## 6. Geometry and Topology

### Euclidean and Non-Euclidean Geometries
Geometry studies shapes, sizes, and properties of space:

- **Euclidean Geometry**: Based on postulates established by Euclid; deals with flat spaces.
  
- **Non-Euclidean Geometry**: Explores geometries based on different axioms (e.g., hyperbolic or spherical geometry).

### Basic Concepts in Topology
Topology studies properties preserved under continuous transformations:

- **Open and Closed Sets**: Fundamental concepts defining neighborhoods around points.
  
- **Homeomorphisms**: Continuous functions with continuous inverses indicating topological equivalence between spaces.

## 7. Probability and Statistics

### Understanding Randomness
Probability theory quantifies uncertainty:

- **Random Variables**: Functions mapping outcomes to real numbers; can be discrete or continuous.
  
- **Probability Distributions**: Describe how probabilities are distributed over values; common distributions include normal, binomial, and Poisson distributions.

### Statistical Inference
Statistical methods draw conclusions from data:

- **Hypothesis Testing**: Procedures to test assumptions about populations based on sample data.
  
- **Confidence Intervals**: Ranges within which population parameters are estimated to lie with a certain probability.
