# Writing a joke

### Set up premise + punch line that breaks the expecatation
- What is the audience expectation?
- Break the audience expectation

### Delivery
- You need a good set up + good delivery.
- Has to flow + good word choice
- Always commit to the joke
- Assume everyone will want to listen

### Improving the joke
- Make it more specific / descriptive - advanced words.
- Cadence - how you write + say the words.
